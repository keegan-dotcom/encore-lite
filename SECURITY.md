# Security Policy

Encore Lite is a read-only skill bundle: it contains no executable scripts,
no MCP servers, and no hooks. It never sends messages, spends money,
publishes, merges, or deletes on the user's behalf.

## Reporting a vulnerability

If you find a security issue in this plugin or in Encore's distribution
pipeline, email **contact@selby.studio**. We aim to respond within 48 hours.
Please do not open public issues for security reports.
