# Encore Lite 🎟

**The end-of-week surprise builder — free edition.**

Claude's weekly usage cap resets on a fixed day per account (check Settings →
Usage) — and whatever you don't use expires with it. The night before yours
resets, Encore reads what you've actually been working on and spends that
leftover capacity on one bonus deliverable you didn't get to — or didn't think
of — in time. An investor gets a research memo. A developer gets a repo audit.
A creator gets a ready-to-post drop. Whoever you are, it builds *your* version
of "one great artifact," grounded in your real work.

## Install

Directly from this repo:

```
/plugin marketplace add keegan-dotcom/encore-lite
/plugin install encore-lite@encore-lite
```

Or attach `skills/encore-lite/SKILL.md` to any Claude chat and save it when
Claude offers.

## Use

- **"run encore"** — one surprise build from your recent work, right now
- **"set up my encore"** — schedules it weekly, for the night before your
  usage cap resets (it asks your reset day once; Sunday evening by default)

Works best with a Claude Project holding your active work — that's the corpus
Encore reads. The more you've connected (files, repos, tools), the more
personal the build.

## What Lite does and doesn't do

Encore Lite reads broadly but acts conservatively: it never sends, publishes,
spends, merges, or deletes anything on your behalf. Its output is a speculative
starting point, clearly labeled — not financial, legal, or professional advice.
