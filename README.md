# Encore Lite 🎟

**The end-of-cycle surprise builder — free edition.**

You paid for a full month of Claude. Most months, some of it goes unused. The
night before your usage resets, Encore reads what you actually worked on all
month and spends that leftover capacity on one bonus deliverable you didn't get
to — or didn't think of — in time. An investor gets a research memo. A developer
gets a repo audit. A creator gets a ready-to-post drop. Whoever you are, it
builds *your* version of "one great artifact," grounded in your real work.

## Install

From the community marketplace:

```
/plugin marketplace add anthropics/claude-plugins-community
/plugin install encore-lite@claude-community
```

Or directly from this repo:

```
/plugin marketplace add keegan-dotcom/encore-lite
/plugin install encore-lite@encore-lite
```

## Use

- **"run encore"** — one surprise build from your month, right now
- **"set up my monthly encore"** — schedules it for the end of each cycle

Works best with a Claude Project holding your active work — that's the corpus
Encore reads. The more you've connected (files, repos, tools), the more
personal the build.

## What Lite does and doesn't do

Encore Lite reads broadly but acts conservatively: it never sends, publishes,
spends, merges, or deletes anything on your behalf. Its output is a speculative
starting point, clearly labeled — not financial, legal, or professional advice.

## The full Encore

The full version adds the four worked playbooks (investor · developer ·
